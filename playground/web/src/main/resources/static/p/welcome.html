Codekvast - Continuous Code Cleansing

What is Codekvast?

Codekvast identifies Truly Dead Code in your Java applications.

Truly dead code is code that no real user has used in X days, where X is defined by you. For some applications, X could be a number of days. For other applications, X could be weeks, months or even years. Only you can tell what X is for your application. X is not a fixed value. Once Codekvast has been running for a while, you can experiment with different values of X.

Use Codekvast in production. It is only there you can learn what parts of your code your real users are using and, more interestingly, not using.

Codekvast works for Java and other JVM languages such as Groovy, Scala, Clojure, Jython and JRuby.

Codekvast is designed to be always on. Codekvast is extremely efficient, and will not disturb normal operation of your application. Codekvast requires very little administration.

Why would I need Codekvast?

Codekvast enables Continuous Code Cleansing, which will

  • Reduce Maintenance Costs. Less code means, well, less maintenance...
  • Increase Agility. You will have less code that slows down development.
  • Improve Quality. Less code means fewer bugs.
  • Improve Security. Less code means fewer vulnerabilities.

How does it work?

  • Codekvast Collector captures invocations of your business methods with extremely low overhead, and periodically hands over the invocation data to the Codekvast Daemon.
  • Codekvast Daemon combines the invocation data with an automatically generated inventory of your installed application code.
  • Codekvast Daemon periodically uploads invocation data plus the code inventory to the cloud-hosted Codekvast Server.
  • Codekvast Server has a real-time web interface for visualizing the captured data. This is where you play around and learn what is your application's Truly Dead Code for different values of X.
  • Once you have identified a chunk of truly dead code, you can go back to your development environment and just delete it!

NOTE: Codekvast does not clean anything automatically! You have to do that yourself.
